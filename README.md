# Barista Koçu Bot

Bu Telegram botu, baristalar için özel olarak hazırlanmış V60, Chemex ve AeroPress demleme tariflerini içerir.

## Kurulum

1. Gerekli kütüphaneyi yükleyin:
   ```bash
   pip install -r requirements.txt
   ```

2. Botu çalıştırın:
   ```bash
   python barista_bot.py
   ```

## Özellikler
- **V60 Tarifi**: Hassas ve dengeli.
- **Chemex Tarifi**: Berrak ve zarif.
- **AeroPress Tarifi**: Pratik ve yoğun.
- **Barista İpuçları**: Her yöntem için profesyonel tavsiyeler.
